# Jargon Progress Repository

A repository to track our progress and store benchmark screenshots as we prepare for the CHPC selection round at UWC.

## Repository Structure

```
Jargon/
├── -Tut-1/                  # Tutorial 1 — member screenshots
├── -Tut-2/                  # Tutorial 2 — member screenshots
├── Btop-From-Source/        # Btop built from source — member screenshots
├── Fastfetch/               # Fastfetch setup — member screenshots
├── Guides/                  # Detailed step-by-step guides for current distro & software versions
├── HPL-Install/             # HPL installation — member screenshots
├── HPL-Optimisation/        # HPL optimisation — member screenshots
├── Passwordless-SSH/        # Passwordless SSH setup — member screenshots
├── Tutorial-Briefs/         # Brief summaries and reference material for each tutorial
├── Documents.lnk            # Shortcut to shared documents
└── README.md
```

## Folders

| Folder | Description |
|---|---|
| `-Tut-1` | Screenshots and work from Tutorial 1 |
| `-Tut-2` | Screenshots and work from Tutorial 2 |
| `Btop-From-Source` | Building and running Btop compiled from source |
| `Fastfetch` | Fastfetch configuration and output screenshots |
| `Guides` | Step-by-step guides for current distro and software versions |
| `HPL-Install` | HPL benchmark installation process and screenshots |
| `HPL-Optimisation` | HPL tuning and optimisation results |
| `Passwordless-SSH` | Setting up passwordless SSH between nodes |
| `Tutorial-Briefs` | Short briefs and reference notes for each tutorial topic |
